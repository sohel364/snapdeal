Synopsis

“Head First Java, 2/Ed (Cover Java 5.0)” makes Java very simple. All those who are new to Java and find it difficult to learn can refer to this book by Kathy Sierra and Bert Bates. This book is ideal for those who aim to learn the language programming.

It offers easy to learn approach educating you on the fundamentals and principles of Java. Some of the many topics covered in this book are Graphical User Interface, Network Connectivity, RMI and Sockets etc. It also covers puzzles and visuals for an added learning approach.   

About the Author

Kathy Sierra and Bert Bates have together co-authored various books like Head First Design Patterns, Head First EJB and Head First Servlets and JSP. Kathy Sierra is one among the founders of the online community of Java Programmers – JavaRanch. Worked with Sun Microsystems as a trainer, Kathy Sierra is a game developer. On the other hand, Bert Bates is involved in Software development for over two decades and is also a consultant in Artificial Intelligence.
